How does TIP work?

Each period, we evaluate the following criteria:
  1. Be an active mainnet validator. Submit proof via our Registration form.
  2. Run cosmoshub-4 and provider with >80% uptime.
  3. Run a testnet infrastructure setup that closely mimics mainnet.
