About Hypha's Testnet Incentives Program

Funded by ATOM Accelerator DAO since Nov 2023

The program

Since November 2023, Hypha has worked with AADAO to give testnet validators the opportunity to earn ATOM for participating on the Hub’s testnet.
This program is meant to incentivize high quality validator performance on the Hub’s testnet on a per-month basis.

We regularly host 60+ of the Hub’s mainnet validators learning about new features, good node operations, and preparing for possible operational challenges that we might face on mainnet.

Validators consistently recognize our program as a great way to learn about gaia-specific features; additional features and tooling such as cosmovisor, cosmwasm, tmptop; and an intense crash course in good node operations and troubleshooting.

The Hub provider testnet is public, meaning that anyone can join, run a validator, and learn about gaia node operations and new features on the Cosmos Hub.
However – only active mainnet validators who meet certain criteria are eligible for incentive payments.

Validator eligibility

Each period, we evaluate the following basic criteria:

  1. Be an active mainnet validator. Submit proof via our registration form (only needs to be completed once, unless your keys change).
  2. Validate all available Top N consumer chains secured by the Cosmos Hub (Neutron and Stride) with > 50% uptime.
  3. Use assigned consumer keys for all available Top N consumer chains (i.e., do not reuse your Hub key).
  4. Run a testnet infrastructure setup that closely mimics your mainnet setup.

If you meet these three criteria (most testnet validators do), you can participate in testnet events and earn points.
At the end of each period, we'll translate those points into USDC and pay you for your testnet work! There are even bonus points for completing all events perfectly and remaining unjailed.

We encourage validators to track these criteria for themselves (especially consumer chain uptime) and monitor them closely as we do not make exceptions for these eligibility criteria.
