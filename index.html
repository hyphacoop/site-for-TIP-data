How does TIP work?

Each period, we evaluate the following criteria:
  1. Be an active mainnet validator. Submit proof via our Registration form.
  2. Validate all available Top N consumer chains secured by the Cosmos Hub (Neutron and Stride) with > 50% uptime.
  3. Use assigned consumer keys for all available Top N consumer chains (i.e., do not reuse your Hub key).
  4. Run a testnet infrastructure setup that closely mimics mainnet.
